# Лабораторная работа № 2

## Сборка

Для запуска необходимо установить

* cmake
* make
* g++

Пример запуска на ubuntu

```sh
sudo apt install cmake make g++
git submodule init && git submodule update
mkdir cmake-build
cd cmake-build
cmake ../
make
./lab_02
```
